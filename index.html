הקובץ מוכן! הנה מה שנבנה: **מבנה הקובץ:** - **Hero** עם כותרת, תיאור קצר ושדה חיפוש חי (מסנן לפי שם מתכון או קטגוריה) - **גריד רספונסיבי** — עמודה אחת במובייל, 2 בטאבלט, 3 בדסקטופ - **Header דביק** עם מספר המתכונים הכולל - **Footer** עדין **כל כרטיס מכיל:** - תמונה מ-Unsplash עם Hover zoom - Badge קטגוריה על התמונה - כותרת + דירוג כוכבים (1-5) - כפתור "צפה במתכון" **לעריכת המתכונים** — פשוט ערוך את מערך ה-`recipes` בתחתית הקובץ: ```js const recipes = [ { id: 1, title: "פסטה קרבונרה", category: "פסטה", rating: 5, image: "https://...", }, // ... ]; ```
